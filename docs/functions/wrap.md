# Function: wrap()

> **wrap**\<`T`\>(`value`): `T`[]

Wraps a value or an array of values into an array.
If the input is already an array, it is returned as is.
If the input is a single value, it is wrapped in an array.

## Type parameters

• **T**

## Parameters

• **value**: `T` \| `T`[]

The value or array of values to wrap.

## Returns

`T`[]

The wrapped array.
