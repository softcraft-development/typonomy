# Function: isObject()

> **isObject**(`value`): `value is object`

Checks if a value is a non-`Array` non-`null` `object`.

## Parameters

• **value**: `unknown`

The value to check.

## Returns

`value is object`

`false` if the value is an `Array` or `null`; `true` if the value is an object; `false` otherwise.
