# Type alias: Some\<T\>

> **Some**\<`T`\>: [`Defined`](Defined.md)\<`T`\> \| [`Defined`](Defined.md)\<`T`\>[]

Represents a type that can be either a single value of type T or an array of type T.
Note that T may never be `undefined`.

## Type parameters

• **T**

The type of value.
