# Function: append()

> **append**\<`T`\>(`array`, `value`): `T`[]

## Type parameters

• **T**

The type of elements in the array.

## Parameters

• **array**: `T`[]

The array to append the value to.

• **value**: `T`

The value to append to the array.

## Returns
