# Function: recordOf()

> **recordOf**\<`K`, `V`\>(`data`?): `Record`\<`K`, `V`\>

## Type parameters

• **K** *extends* `PropertyKey`

• **V**

The type of the values in the object.

• **data?**: `Record`\<`K`, `V`\>= `undefined`
