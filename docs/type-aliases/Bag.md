# Type alias: Bag\<T\>

> **Bag**\<`T`\>: [`Optional`](Optional.md)\<[`Some`](Some.md)\<`T`\>\>

Zero, one, or more values of type `T`.
Zero values are represented as `undefined`.
One value is represented as `T`.
More than one value is represented as `Array<T>`.

## Type parameters

• **T**

The type of value.
